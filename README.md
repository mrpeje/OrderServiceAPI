Asp.net Web API with DB on Postgres.

**Requirements**
- Postgres SQL Server (port 5432)

**To run service**
- Build project
- Run OrdersService\bin\Debug|Release\net6.0\OrdersService.exe

example of request URL https://localhost:7125/api/orders
